# GSM Socket KiCad

This project presents the **PCB design of a GSM socket** using **KiCad**.  
It includes the complete KiCad project files, schematics, and PCB layout, suitable for GSM module integration in embedded systems and IoT devices.

### Tools Used
- KiCad EDA
- GSM Socket Module

### Project Contents
- KiCad project folder (.pro, .sch, .kicad_pcb, etc.)
- PCB layout files
- Schematic files
